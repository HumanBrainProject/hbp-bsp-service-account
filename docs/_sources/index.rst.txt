HBP - Brain Simulation Platform Service Account
===============================================

The Service Account of the Human Brain Project (HBP) Brain Simulation Platform 
(BSP) is a rest API service which allows developers to submit jobs, on behalf 
of |HBP_Collab_link| users, on remote HPC systems from within the HBP 
Collaboratory workspace.
