Job Delete/Cancel¶

All the requests reported below must be appended to the service endpoint:

https://bspsa.cineca.it

The allowed HPC string values are:

  • pizdaint
  • nsg

The allowed Project string values are:

  • bsp_pizdaint_01
  • bsp_nsg_01

Deleting a job from a HPC system is perfomed through a DELETE request removes all the output files and related info concerning that job, on the HPC.

Canceling a job is also performed throuh a DELETE request and allows to cancel a job before it is terminated. In this case, the user’s resources which have not been used will be restored.

URL: /jobs/(string: hpc)/(string: project_name)/(string: job_id)/

Required Headers:

  • Authorization:

    Authorization: Bearer <token>

Example:

  • Delete a job

    Example Request

    DELETE /jobs/nsg/bsp_nsg_01/NGBW-JOB-NEURON75_TG-E1B7ADD55CB54F1B9468EEEDFE9F6A64                  HTTP/1.1
Authorization:              Bearer Token
