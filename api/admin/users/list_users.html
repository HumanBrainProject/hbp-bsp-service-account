List Users API allow admin to fetch a list of all users.

URL: /admin/users/

Required Headers:

  • Authorization:

    Authorization: Bearer <token>

Example:

  • Get all users

    Example Request

    GET /admin/users/           HTTP/1.1
Authorization:              Bearer FakeToken

    Example Response

    HTTP/1.1                    200 OK
Content-Type                application/json
